EMR Helper - 빠른 시작 가이드
==================================================

배치 파일이 작동하지 않는 경우:

1. Windows 검색 → "cmd" 입력 → Enter
2. 검은 화면(명령 프롬프트)에서 다음 입력:

cd /d D:\asanhelper
python main.py

3. "OCR 모듈 로딩 중..." 메시지가 나오면 성공!
4. Ctrl+Shift+A로 테스트

==================================================
참고:
- 이 방법은 Python이 설치된 PC에서만 작동
- .exe 빌드는 복잡하므로 일단 이 방법으로 테스트
- Python 필요: python.org에서 다운로드
